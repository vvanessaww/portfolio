# Portfolio

A minimalist 3D portfolio site featuring a desk setup with interactive artifacts.

## Vision
- ThreeJS home screen with a desk environment
- Interactive 3D objects (laptop, postcard, notebook, etc.)
- Each artifact is clickable and links to portfolio projects
- Clean, minimalist aesthetic

## Stack
- React + Vite
- Three.js / React Three Fiber
- React Router for navigation

## Projects Featured
- Postcard → Strava Postcard generator
- Laptop → Code projects / GitHub
- Notebook → Blog / Writing
- (More to be added)

## Development
```bash
npm install
npm run dev
```
